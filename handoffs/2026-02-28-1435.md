## GOTCHAS DISCOVERED
- **TheOddsAPI futures cost 2×**: Outrights endpoints billed at 2 requests each. 3 markets = 6 req/run. Documented in agent and CLAUDE.md.
- **422 on unavailable markets**: Futures endpoints return 422 (not 404) when no odds are posted. Agent skips gracefully.
- **Value direction counter-intuitive**: Odds getting SHORTER = team more favored = bettor's ticket worth MORE. `direction === 'better'` when `currentDecimal < entryDecimal`. ArrowDownRight icon is intentional.
- **Auto-save guard anti-pattern**: `if (state.length > 0) saveToStorage(...)` prevents cleared state from persisting across refresh. Remove all guards; use explicit clearStorage() in clear handlers.
- **Boot clobber**: Unconditional `setSplits(remoteData || {})` in boot effect overwrote user data on every refresh (and nuked it on network failure). Fix: always check localStorage first.
